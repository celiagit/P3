Hikari

Hikari is a house-shaped smart bookshelf with a built-in lamp. This smart bookshelf allows users to read the book in their bed before they go to sleep without turning on the light of the lamp.

Project Description

Hikari encourages users to read books. There is a light under the shelf which could sense the amount of light in the room it's in, and lights up automatically when users start reading. The two sides of the bookshelf where users can put their favorite books. Also, the users can control the location of the light and check how long they spend time reading as well as brightness of the light using a mobile networking application Blynk.

Process & challenge


In starting this project, I and my group members discussed a long time to determine the shape of the bookshelf and what kind of sensor we could use to improve the usability. In this project, we used three kind of sensors, which are light sensor, proximity sensor, and Pan-tilt motor. The main function of this bookshelf is it can interactive with the light. I considered a lot on the shape of the bookshelf. For example, it could hide all circuits; it needs to be able to withstand weight steadily; it must be small and pretty because it will be placed on the headboard. That's why the shape of the bookshelf is an asymmetrical house.

At the beginning , we wanted to use proximity sensor for measuring the distance between people and the shelf. But we found that some small movements of people will also affect the sensitivity of the sensor, which will make it flash from time to time. So we changed the sensor put on the "roof" of the bookshelf. This roof can act as a large bookmark so that both books can be placed and lights can be controlled.
